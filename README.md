# Datas CMS

## Installation

Clone repositoryet:

```bash
git clone https://github.com/datasdk/foundation-backend-api
cd foundation-backend-api
```

Installer PHP dependencies:

```bash
composer install
```

Run the Datas installation:

```bash
php artisan datas:install
```

When the installation is complete, the project is ready for configuration and use.

# Datas Module

This repository contains a module used within the Datas ecosystem.

## Usage

This module is not intended to run as a standalone application.
It is designed to be installed and used through the Datas platform.
